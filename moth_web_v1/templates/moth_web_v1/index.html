Getting Started

Here is what you need to do:

  1. Register your team. This only needs to happen once per team, so if somebody else on your team has already done it, you don't need to.
  2. Get an overview of puzzles, and start working on something. The list of open puzzles changes over time, you need to reload the page to get the current version!
  3. Check the scoreboard in another tab, to see how your team is doing.
